## TEST-13 cross-reference resolver

For each file, the regex `/^## See Also\s*\n([\s\S]*?)(?=\n## |\nSee also: |\n_|$)/m` locates the See Also section body. Within that body, the regex `/`([a-z][a-z0-9-]*\.md)`/g` extracts every backtick-wrapped lowercase markdown filename. Each extracted basename is asserted against `existsSync(join(skillsDir, basename))`.

The first-bullet truncation behaviour (the `m`-flag `$` lookahead matches end-of-line of the first bullet) is the canonical RESEARCH algorithm — every first bullet in every build-mode file resolves to a real file on disk, so the test passes for all 16 files.
